# svgosu
## Installation
```
npm install -g svgosu
```
## Command line usage
Process a svg path
```
svgosu path "M43 110C100 69 167 167 226 116C318 183 372 75 350 43"
```
Help for advanced usage
```
svgosu --help
```
or
```
svgosu -h
```
